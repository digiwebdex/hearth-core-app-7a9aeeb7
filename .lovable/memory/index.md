# Project Memory

## Core
Self-hosted VPS (Hostinger). No Lovable Cloud/Supabase. Backend = Node.js + Express + Prisma + PostgreSQL.
Multi-tenant SaaS. Frontend = React + Vite + Tailwind. API at VITE_API_URL env var.
